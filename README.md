
# Create: Motorsport

A NeoForge 1.21.1 add-on for Create, focused on bringing motorsport-themed parts and systems into Minecraft.

## Development

- Minecraft: `1.21.1`
- NeoForge: `21.1.233`
- Create: `6.0.10`
- Java: `21`

Common commands:

```powershell
.\gradlew.bat build
.\gradlew.bat runClient
.\gradlew.bat runData
```

The mod currently registers a starter `racing_component` item and a `Create: Motorsport` creative tab.
